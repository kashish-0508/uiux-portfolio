# Kashish Bhardwaj — UI/UX Designer Portfolio

A creative and responsive personal portfolio website showcasing my UI/UX design work, design process, skills, and experience.

## About

I am a UI/UX Designer with a B.Tech in Computer Science. I focus on creating clear, visually engaging, and user-friendly digital experiences for mobile apps, responsive websites, and landing pages.

This portfolio highlights three selected projects:

- **Study Hub** — Student Productivity Mobile App
- **Prep AI** — AI Interview Preparation Landing Page
- **Luméra** — Luxury Skincare Clone Landing Page

## Features

- Creative and responsive portfolio layout
- Smooth scroll reveal animations
- Interactive hover effects
- Magnetic button interactions
- Animated marquee section
- Custom cursor glow effect
- Responsive mobile navigation
- Project image previews
- “View in Figma” buttons for project case studies
- Contact and LinkedIn links

## Run Locally

No installation or build process is required.

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

For a better development experience, you can also run the project with the Live Server extension in Visual Studio Code.

## Customization

To update the portfolio:

- Replace images inside the `assets` folder.
- Add your real Figma project links to the “View in Figma” buttons in `index.html`.
- Update project descriptions directly in `index.html`.
- Change colors from the CSS variables at the top of `style.css`.
